QUICK START

Step 1) clone the repo 
git clone https://github.com/afvignesh/sugarbox_assignment.git

Step 2) go to the directory "Sugarbox_assignment" and run command
 --> go get
 --> go build
 --> ./sugarbox

Step 3) Go to browser and enter
localhost:8000/api
